Knesset’s data used to be a complete mess. IsraParlTweet1 is a vast database of Knesset quotes and MKs’ tweets from the past ~30 years. Accessing the data through a database doesn’t necessarily reveal trends, consistent opinions, or meaningful insights – you must sift through an enormous amount of textual information that isn’t easy to analyze.

Knessight uses advanced AI tools to extract and analyze political positions from speeches and tweets of Israeli MKs. It assigns stance scores on key topics and visualizes trends, correlations, and changes over time, and provides you with easy access to them.

In order to extract the relevant data for a specific topic concerning an individual Member of Knesset from quotations, we follow this 4-stage approach:

Raw Text – Retrieving all records from the IsraParlTweet database that were spoken by the MK.

Filter – Using a lightweight LLM model that assigns a score between 1 and 5 based on how relevant each quote is to the topic, and filtering out sentences that scored below 4.

Score – Applying a more advanced LLM model to assign a score between 1 and 10, indicating how positive each relevant quote is toward the topic.

UI – Storing the results in an accessible user database and providing an inviting and interactive UI for viewing the analysis.

What's next? There's still a lot of work ahead: analyzing more and more MKs, diving into new topics, improving the model to make it as efficient, fast, and accurate as possible, evaluating its precision to refine it, creating more insightful data-driven graphs, and presenting users with valuable insights — both reinforcing existing knowledge and uncovering new perspectives they may not have considered.
